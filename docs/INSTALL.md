# Install skool-view (Firefox)

Niet van toepassing op deze fork. Deze fork bouwt en gebruikt alleen de **Chrome**-variant — zie
[`INSTALL-chrome.md`](INSTALL-chrome.md).

Firefox-ondersteuning uit het originele project ([rocleemusic/skool-view-dist](https://github.com/rocleemusic/skool-view-dist))
zit nog in de broncode (`manifest.json`'s `{{firefox}}`-secties), maar wordt in deze fork niet
gebouwd of getest.
